# 🛒 E-commerce App

A full-stack e-commerce application where users can browse products, manage their cart, and place orders.

## 🚀 Live Demo
https://metin-ecommerce.vercel.app

## ⚙️ Tech Stack
- React (Vite)
- Node.js & Express
- REST API

## ✨ Features
- Product listing
- Add/remove items from cart
- Update delivery options
- Order creation
